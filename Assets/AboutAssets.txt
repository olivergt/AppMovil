Cualquier activo sin procesar que desee implementar con su aplicación se puede colocar en
este directorio (y los directorios secundarios) y se le da una acción de compilación de "AndroidAsset".

Estos archivos se implementarán con su paquete y se podrá acceder a ellos mediante Android
AssetManager, así:

public class ReadAsset : Activity
{
    protected override void OnCreate (Bundle bundle)
    {
        base.OnCreate (bundle);

        InputStream input = Assets.Open ("my_asset.txt");
    }
}

Además, algunas funciones de Android cargarán automáticamente archivos de recursos:

Typeface tf = Typeface.CreateFromAsset (Context.Assets, "fonts/samplefont.ttf");
